Working Memory Model: Truth vs Deception
TRUTH-TELLERS
Primary Source
Specific experiences & memories
Content Characteristics
Schema-inconsistent details, irrelevant information
Memory Decay
Recall becomes less detailed over time
LIARS
Primary Source
General schemas & templates
Content Characteristics
Schema-consistent details, coherent narrative
Consistency
Stories remain consistent over time
Deception = Memory Recombination
Not pure invention, but recombining existing memories and schemas
Based on Hume's (1748/2007) conceptualization
Key Predictions:
• Liars → more schema-consistent details
• Truth-tellers → more schema-inconsistent/irrelevant details
• Memory decay affects truth-tellers' recall over time
Baddeley (2007); Sporer (2016)
